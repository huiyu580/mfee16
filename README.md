# Elfin 音樂餐廳前台 
### MFEE16 第四組
成員： 羅芳奕、簡以捷、吳奕勳、李蕙伃、葉思涵、詹宜諺
  
  
## 網站功能
提供顧客線上訂位、外送訂餐、音樂表演直播、歌手投票、撰寫評論、小遊戲、會員註冊等服務。
  
  
## 使用技術
前端： React JS
後端： Node JS / express框架
資料庫：　MySQL

## Git Flow 協作

撰寫評論 (comment 分支) - 葉思涵
外送訂餐 (delivery 分支) - 羅芳奕
